# Pig-dice-gam 🎲

A fun dice game built using Python where players try to reach the target score by rolling a dice and making strategic decisions.

## Features
- Multiplayer gameplay
- Random dice rolling
- Score tracking for each player
- Risk and reward mechanics
- Winning condition based on target score

## Technologies Used
- Python
- Random Module
- Loops
- Conditional Statements
- Functions

## Learning Outcomes
- Working with random number generation
- Using loops and conditions
- Managing player scores
- Building interactive games in Python
- Understanding game logic

## How to Run
1. Open the project in Python.
2. Run the program.
3. Enter the number of players.
4. Roll the dice or hold your score during your turn.
5. The first player to reach the target score wins.

## Game Rules
- Players take turns rolling a dice.
- If a player rolls a 1, their turn ends and they lose the points earned in that turn.
- Players can choose to hold and add their turn score to their total score.
- The first player to reach the target score wins the game.
